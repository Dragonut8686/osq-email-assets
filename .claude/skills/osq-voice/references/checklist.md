# Чеклист самопроверки текста (прогонять перед показом пользователю)

Отвечай честно по каждому пункту. Любой провал = переписать до показа.

1. **Первое знакомство.** Каждый продукт понятен человеку, который видит его
   впервые? Названы формат, блюда и каналы продаж?
2. **Формула блока.** В каждом продуктовом блоке есть связка
   «что это → как работает на бизнес → факт → выгода»?
3. **Запретный словарь.** Ни одного слова из банлиста `lexicon.md`
   (инновационный, уникальный, данный, является, «в современном мире»…)?
4. **Только подтверждённые факты.** Все цифры (мл, мм, °C, шт, кг) взяты
   из материалов клиента, ничего не выдумано?
5. **Проза, не буллеты.** Продуктовые блоки написаны связными абзацами
   по 2–4 предложения? Буллеты — только в карточках характеристик?
6. **Названия продуктов** — латиницей капсом точно по лексикону
   (OSQ PILLOW XL, BLACK WOOD…)?
7. **Эмодзи** — только в title/прехедере (1–2), в теле письма их нет?
8. **Внутренней кухни нет**: ни «уточним», ни «placeholder», ни ссылок
   на процесс подготовки?
9. **CTA** — глагол действия 2–3 слова из набора формул, у финального блока
   есть приглашение к диалогу?
10. **Длина под зум.** Абзацы короткие (2–4 предложения), заголовки ёмкие —
    текст выживет при сжатии до мобильной ширины?
11. **Тон OSQ**: экспертно, конкретно, тепло — без давления и восклицаний
    в каждой фразе?
12. **Дубли.** Одна и та же мысль не повторяется в бейдже, заголовке и абзаце?
